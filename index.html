  • واجهة سهلة الاستخدام

النص المستخرج

مرحبا، هذا نموذج لمحاكاة عملية تحويل الصوت إلى نص. هذه التقنية تستخدم خوارزميات الذكاء الاصطناعي لتحويل الكلام المنطوق إلى نص مكتوب. يمكن استخدام هذه التقنية في العديد من التطبيقات مثل التسجيلات الصفية، تحليل المشاعر، والمساعدات الصوتية.

إعداد مفاتيح API

لبدء استخدام المشروع، تحتاج إلى إعداد مفاتيح API من الخدمات التالية:


# إعداد ملف .env
COHERE_API_KEY=your_cohere_key_here
GOOGLE_APPLICATION_CREDENTIALS=path_to_your_google_cloud_credentials_json

# تشغيل إعداد Google Cloud
python setup_gcloud.py

# تحويل ملف صوتي إلى تنسيق 16kHz WAV
ffmpeg -i input.mp3 -ar 16000 -ac 1 audio_input.wav
